Gerador de Widget contador de views para Kick

Digite o nome do seu canal e receba o link para usar no OBS.

Como usar no OBS:
  1. Copie o link gerado abaixo.
  2. No OBS, adicione uma nova fonte do tipo Navegador (Browser Source).
  3. Cole o link copiado no campo de URL da fonte.
  4. Defina a largura para 300 e altura para 100 (ou ajuste conforme desejar).
  5. Clique em OK para adicionar o widget à sua transmissão.
